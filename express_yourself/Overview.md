Parse
=====

```
parse "x + 0"               ~~> Add(Var "x", Const 0)
parse "x      +          0" ~~> Add(Var "x", Const 0)
parse "x + 0 * 5"           ~~> Add(Var "x", Mul(Const 0, Const 5))
parse "x + (0 * 5)"         ~~> Add(Var "x", Mul(Const 0, Const 5))
parse "x + ((0 * (5)))"     ~~> Add(Var "x", Mul(Const 0, Const 5))
parse "(x + 0) * 5"         ~~> Mul(Add(Var "x", Const 0), Const 5)

scan "(x + 0 )  * 5"  ~~> [ LPar; Id "x"; Plus; Num 0; RPar; Mult; Num 5 ]
scan "(x + 0)*5"      ~~> [ LPar; Id "x"; Plus; Num 0; RPar; Mult; Num 5 ]

```

Grammar
-------

```
E  ::= Num
     | Id
     | E '+' E
     | E '-' E
     | E '*' E
     | E '/' E
     | '(' E ')'
     | 'if' E 'then' E 'else' E
     | 'let' Id '=' E 'in' E
```

```
E    ::= E '+' T | E '-' T | T
T    ::= T '*' F | T '/' F | F
F    ::= Num | Id | '(' E ')'
```

```
E     ::=  T  Eopt
Eopt  ::=  '+' T Eopt | '-' T Eopt | ε
T     ::=  F Topt
Topt  ::=  '*' F Topt | '/' F Topt | ε
F     ::=  Num | Id | '(' E ')'
```
